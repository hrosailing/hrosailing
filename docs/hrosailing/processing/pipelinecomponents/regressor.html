Module hrosailing.processing.pipelinecomponents.regressor

Contains the baseclass for Regressors used in the CurveExtension class, that can also be used to create custom Regressors.

Also contains two predefined and usable regressors, the ODRegressor and the LeastSquareRegressor.

The function which describes the model and is to be fitted.
