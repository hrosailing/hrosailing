Module hrosailing.pipelinecomponents.regressor

Classes used for modular modeling of different regression methods

Defines the Regressor Abstract Base Class that can be used to create custom regression methods

Subclasses of Regressor can be used with the CurveExtension class in the hrosailing.pipeline module

class LeastSquareRegressor (model_func: Callable, init_vals=None)

A least square regressor based on scipy.optimize.curve_fit

Parameters

model_func : function or callable

The function which describes the model and is to be fitted.

The function signature should be f(ws, wa, *params) -> bsp, where ws and wa are numpy.ndarrays resp. and params is a sequence of parameters that will be fitted

init_vals : array_like ,optional

Inital guesses for the optimal parameters of model_func that are passed to scipy.optimize.curve_fit

Defaults to None
